# SilCor Tech · Soluciones Digitales

**Ayudamos a negocios que se animan a lo digital.**

Somos dos programadores recibidos que decidimos emprender juntos. Trabajamos con negocios de cualquier rubro que quieren sumar tecnología a su día a día — desde una web institucional hasta un sistema propio — con precio cerrado y trato directo.

🔗 **Sitio en vivo:** https://silcortech.vercel.app/
📸 **Instagram:** [@SilCorTech](https://instagram.com/SilCorTech)

---

## ✨ Qué hacemos

- **Web institucional** — presencia profesional online
- **Tienda / catálogo online** — pedidos por WhatsApp o pago online
- **Sistemas a medida** — turnos, reservas, gestión interna
- **Integraciones y mantenimiento** — conectamos herramientas existentes y damos soporte continuo

## 🧭 Cómo trabajamos

1. **Diagnóstico** — charlamos sobre tu negocio y vemos qué necesita primero
2. **Propuesta** — te mostramos un plan y precio cerrado, sin sorpresas
3. **Desarrollo y entrega** — construimos, testeamos y lanzamos, con seguimiento post-entrega

## 🛠️ Stack técnico

- [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- Deploy en [Vercel](https://vercel.com/)

## 🚀 Cómo correr el proyecto localmente

Clonar el repositorio:

    git clone https://github.com/NachoCorsini/SilCorTech.git

Entrar a la carpeta:

    cd SilCorTech

Instalar dependencias:

    npm install

Correr en modo desarrollo:

    npm run dev

El proyecto va a estar disponible en `http://localhost:5173`.
